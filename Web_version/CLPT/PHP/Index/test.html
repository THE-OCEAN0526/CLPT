CONCEPT

使程式設計變得更容易

程式語言作為一種標準化的溝通工具，用於向電腦發送指令，是一種使程式設計師能準確定義電腦所需資料的語言。 儘管自然語言和程式語言在功能上有著重大差異，但兩者都需要具備強大的表達能力和簡潔明了的機制。 我們的想法是創建一個讓使用者能以自然語言與電腦進行交流的平台。 我們因此開發了一款中文口述程式語言轉換器，讓使用者能夠將語言轉換成程式碼，藉此實現更直觀的程式創作方式。
